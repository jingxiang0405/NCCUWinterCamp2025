## nccu_cscamp
### Description

輸入一個整數 $N$ ，代表輸出的次數
輸出$N$行 "nccu_cscamp"

### Input

輸入一個整數 $N$ $(N \ge 1)$

### Output

輸出 $N$ 行 "nccu_cscamp"

### Sample Input 1 
```
3
```

### Sample Output 1
```
nccu_cscamp
nccu_cscamp
nccu_cscamp
```
